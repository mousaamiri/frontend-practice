# Filterable Product List

فیلتر، جست‌وجو و مرتب‌سازی زنده‌ی یک لیست محصولات — نزدیک‌ترین تمرین به رندر پویای لیست‌ها در React.

## مفاهیم تمرین‌شده

- **جداسازی داده‌ی ثابت از state متغیر** — آرایه‌ی `products` هیچ‌وقت در state نیست، چون خودش تغییر نمی‌کند؛ فقط فیلترها (`searchTerm`, `selectedCategory`, `sortBy`, `inStockOnly`) در state نگه‌داری می‌شوند
- **زنجیره‌ی filter/sort مبتنی بر state** — یک تابع (`getFilteredProducts`) تمام شرط‌های فعال را روی کپی‌ای از داده‌ی اصلی اعمال می‌کند
- **استخراج مقادیر یکتا با Set** — ساخت لیست دسته‌بندی‌ها مستقیماً از داده، بدون نوشتن دستی
- **جلوگیری از mutation آرایه‌ی اصلی** — کپی گرفتن (`[...products]`) قبل از `sort`، چون `sort` آرایه‌ی اصلی را نیز تغییر می‌دهد
- **مقداردهی یک‌بار در برابر رندر مکرر** — پر کردن گزینه‌های دسته‌بندی فقط یک‌بار در شروع برنامه انجام می‌شود، نه در هر فراخوانی `render()`، چون این بخش از UI به state وابسته نیست

## نکته‌ی کلیدی

State این پروژه فقط شامل «چیزهایی که واقعاً تغییر می‌کنند» است، نه خود داده:

```js
let state = {
    searchTerm: "",
    selectedCategory: "all",
    sortBy: "name",
    inStockOnly: false
}
```

این تفکیک — داده‌ی ثابت جدا از state تعامل کاربر — همان تصمیمی است که در کامپوننت‌های React هم مدام گرفته می‌شود.
