در این بخش با ساختار اولیه زبان سی‌شارپ، متغیرها، انواع داده‌ها، دستورات شرطی (if-else) و حلقه‌های تکرار (for و while) به همراه حل تمرین‌های عملی آشنا می‌شوید.

این فصل به آموزش مفاهیم کلیدی شی‌گرایی شامل کلاس‌ها (Classes)، اشیاء (Objects)، ارث‌بری (Inheritance)، چندریختی (Polymorphism) و کپسوله‌سازی (Encapsulation) می‌پردازد.

در این بخش، نحوه اتصال برنامه به دیتابیس SQL Server، اجرای کوئری‌ها، کار با LINQ و مدیریت داده‌ها با استفاده از فریم‌ورک محبوب Entity Framework Core را یاد می‌گیرید.
