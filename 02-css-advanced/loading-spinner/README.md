# Loading Spinner / Skeleton Loading

Three common loading-state patterns built with pure CSS `@keyframes` animations: a rotating spinner, an indeterminate progress bar, and a skeleton screen with a shimmer effect.

## Preview

![Loading Spinner Preview](docs/preview.gif)

## What I Practiced

- `@keyframes` with multiple stops (not just `transition`)
- `border` trick for a circular spinner (`border-bottom-color` different from the rest)
- Shimmer effect using an animated `background-position` on a gradient
- Building three distinct loading patterns in one exercise
