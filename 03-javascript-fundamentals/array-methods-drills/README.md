# Array Methods Drills

تمرین چهار متد پرکاربرد آرایه در JS مدرن: `map`, `filter`, `reduce`, `find`.
پایه‌ی طرز فکر declarative که در رندر کردن لیست‌ها در React استفاده می‌شه.

## مفاهیم تمرین‌شده

- **map** — ساخت آرایه‌ی جدید هم‌طول با تبدیل هر عنصر (استخراج فیلد، اضافه کردن property، فرمت کردن رشته)
- **filter** — انتخاب زیرمجموعه بر اساس شرط (تک‌شرطی، ترکیبی، و زنجیره با map)
- **find** — پیدا کردن اولین عنصر منطبق با شرط (برمی‌گردونه تک آبجکت یا `undefined`، نه آرایه)
- **reduce** — جمع‌بندی آرایه به یک مقدار (جمع، max/min، میانگین، شمارش شرطی، گروه‌بندی به object)

## نکته‌ی کلیدی

ترتیب متدها در زنجیره مهمه:

```js
products.filter(p => p.price > 10).map(p => p.name)   // ✅ درست کار می‌کنه
products.map(p => p.name).filter(p => p.price > 10)   // ❌ خالی برمی‌گرده، چون بعد از map دیگه item یه string ساده‌ست نه object
```
